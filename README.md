## 📌 Overview

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/May12365/SoftwareDevelopmentForCrossPlatform-.git
```

### Enter Project Folder

```bash
cd SoftwareDevelopmentForCrossPlatform-
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```

---
